如何操作：
使用gcc編譯 輸入指令：gcc B_plus_tree.c -o tree
執行執行檔 輸入指令：./tree
(gcc version 9.2.0)

[其他事項]
(1) 預設每一個node的 entry數量為 4，此設定在開始的 #define定義，若需要更動則需要直接更改程式碼
(2) 在選擇 display 的時候，會顯示出 in-order 序列，顯示出樹的順序，接下來有數行縮排的結果，為以表示樹的實際長相。
其中第一行代表root中所有index的值，之後每縮排一次代表進入下一層，同一排的順序為B+ tree中從左到右依序顯示。若有以<>包住的數列，則代表其為葉節點(實際舉例可見附圖)
(3) 以縮排表示葉節點時，皆是按照順序，若不滿5個葉節點，則代表該葉節點不存在，若是裡面的內容皆被刪除，則會只顯示<>
